# CMake?
Makefile의 루틴을 추상화해서 빌드 과정을 읽기 쉽게 기술할 수 있는 도구

# 주요 명령어
## set
변수를 정의  
> ### 정의
> ```cmake
> set(<변수명> <값>)
> ```
> ### 목록 변수 정의
> ```cmake
> set(<변수명> <항목1> <항목2> ... <항목n>)
> ```
> ### 변수 참조
> ```cmake
> $변수명
> ${<변수명>}
> ```
## add_executable
빌드 최종 결과물로 생성할 실행 파일을 추가
```cmake
add_executable(<실팽 파일명> <소스 파일1> <소스 파일2> ... <소스 파일n>)
```
> ### 예
> ```cmake
> add_executable(app.out main.c foo.c bar.c)
> ```
